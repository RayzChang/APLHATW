## 在開始之前，先搞懂幾個詞

如果你是股市新手，這些詞會反覆出現，先花 2 分鐘認識它們：

| 詞彙 | 白話解釋 |
|------|---------|
| **股票代號** | 每支股票的身分證號碼。比如「2330」就是台積電、「2317」就是鴻海。在台灣，通常是 4 位數字。 |
| **買入 (BUY)** | 花錢買股票，你覺得它會漲。 |
| **賣出 (SELL)** | 把手上的股票賣掉，換回現金。 |
| **觀望 (HOLD)** | 先不動，等更好的時機。 |
| **停損** | 股票跌到某個價格時，認賠賣掉。目的是「小賠」，避免「大賠」。就像打牌知道牌爛就蓋牌，不要硬拚。 |
| **停利** | 股票漲到某個價格時，獲利了結賣掉。目的是「落袋為安」，避免漲了又跌回來。 |
| **持倉** | 你目前手上持有的股票。「目前持倉 3 檔」意思是你同時買了 3 支不同的股票。 |
| **模擬交易** | 用假的錢買賣股票。帳面上的數字都是虛擬的，你不會真的花到錢，也不會真的賺到錢。目的是先練習、先驗證。 |
| **回測** | 把時間倒轉，用過去的歷史資料模擬「如果那時候這樣買賣，結果會怎樣」。類似考試前先寫模擬考卷。 |
| **勝率** | 10 次交易裡面，幾次是賺的。勝率 60% 意思是 10 次裡面有 6 次賺錢。 |
| **報酬率** | 你賺了本金的百分之幾。100 萬本金、報酬率 +5% 代表你賺了 5 萬。 |
| **K 線** | 股票每天的價格走勢圖，長得像一根一根的蠟燭。紅色通常代表漲、綠色代表跌（台灣的規則，跟美國相反）。 |
| **ATR** | 一支股票平均每天上下波動多少錢。波動大的股票 ATR 高，波動小的 ATR 低。系統用 ATR 來決定停損停利要設多遠。 |

## 這個軟體到底在幹嘛？

一句話：

> **讓 AI 幫你看盤、選股、模擬買賣，你只要決定要不要跟著做。**

想像你請了一個 24 小時不睡覺的股票分析師，他每天早上自動幫你：

1. 掃描台股 2,000 多支股票
2. 用數學公式 + AI 智慧挑出最有機會漲的
3. 用虛擬的 100 萬幫你模擬買進
4. 每分鐘盯盤，該賣就賣

你要做的只有一件事：**看 AI 買了什麼，決定你要不要用真錢跟著買。**

## 功能一：AI 模擬交易（看 AI 怎麼操盤）

### 在哪裡？

打開網站，預設就是這個頁面。

### 畫面上有什麼？

從上到下依序是：

#### 1. 資產總覽（最上面的數字欄）

```
💰 虛擬總資產    💵 可用現金    📈 累積總獲利    🎯 綜合報酬率
$1,084,030      $500,000     +$84,030       +8.40%
```

- **虛擬總資產**：AI 幫你管的錢，包含現金 + 持有的股票市值
- **可用現金**：還沒拿去買股票的錢
- **累積總獲利**：從 100 萬開始到現在，總共賺了（或虧了）多少
- **綜合報酬率**：賺了百分之幾

> 這些都是虛擬的錢！不是你的真錢。

#### 2. 操作按鈕列

- **「執行 AI 交易分析（全市場）」**：手動叫 AI 立刻去掃描全市場。平常 AI 會在 09:10 和 12:30 自動掃，但你也可以隨時手動觸發。
- **「刷新數據」**：重新讀取最新的帳戶狀態。
- **「重置模擬」**：把所有模擬紀錄清空，資金歸零重來。（小心！無法復原）

#### 3. AI 自動交易系統說明（藍色區塊）

這個區塊告訴你 AI 目前的運作狀態：

- **上次掃描**：AI 最近一次掃描全市場是什麼時候
- **掃描股票**：這次掃了幾支
- **技術篩選**：從 2,000 支裡面挑出幾個候選
- **AI 下單**：最後真正下了幾筆模擬單

#### 4. 績效走勢圖（左邊）+ AI 決策日誌（右上）+ 即時觀察清單（右下）

- **走勢圖**：你的虛擬資產每天的變化，可以看到是在賺還是在虧
- **AI 決策日誌**：AI 每一筆買賣的紀錄，綠色是買入、紅色是賣出
- **即時觀察清單**：幾支代表性股票的即時行情（台積電、鴻海等）

#### 5. 當前持倉庫存（表格）

AI 目前手上持有哪些股票，每一檔的：
- 買入均價、現在股價
- 目前賺了還是賠了（未實現損益）
- 停損價、停利價設在哪裡
- 持有幾天了

#### 6. AI 回測模擬（可展開區塊）

點擊展開後可以進行歷史回測，下面會詳細說明。

#### 7. 歷史交易紀錄（最下面）

所有已完成的買賣紀錄，包含每一筆的損益。

### AI 是怎麼決定買哪支股票的？

分成兩步：

**第一步：快速篩選（免費，全自動）**

AI 下載全台灣 2,000 多支股票最近幾個月的走勢，然後用 7 個數學指標幫每支股票打分數：

| 指標 | AI 看什麼 |
|------|----------|
| RSI | 這支股票最近是不是被賣太多了（超賣 = 有反彈機會） |
| KD | K 線跟 D 線有沒有交叉（黃金交叉 = 可能要漲了） |
| MACD | 動能方向是往上還是往下 |
| 收盤價 vs 20 日均線 | 股價有沒有站在平均線上面（站上 = 偏多） |
| 5 日均線 vs 20 日均線 | 短期趨勢有沒有向上 |
| 布林通道 | 股價在通道的哪個位置（靠近底部 = 相對便宜） |
| 成交量 | 今天有沒有比平常更多人在買賣（量大 = 有人在關注） |

7 個指標投票，得分 >= 3 分的才進入下一關。

**第二步：AI 深度分析（前 30 名）**

通過第一關的最多 30 支股票，會交給四位 AI 探員做更深入的分析：

```
📈 技術分析師 → 看走勢圖、找型態
         ↓
🎭 情緒分析師 → 看最新新聞、社群風向
         ↓
🛡️ 風控師 → 算風險、設停損停利
         ↓
👔 首席決策官 → 綜合以上，做最終決定
         ↓
    信心度 >= 70% 且判定「買入」→ 下單！
```

只有首席決策官說「買」，而且信心超過 70%，才會真的買進。

### AI 什麼時候會賣掉股票？

| 情況 | 觸發條件 | 白話說明 |
|------|---------|---------|
| **停損** | 虧損超過 2 倍 ATR | 「虧太多了，趕快跑」 |
| **停利** | 獲利超過 3 倍 ATR | 「賺夠了，落袋為安」 |
| **保本** | 先漲超過 1.5%，又跌回成本附近 | 「至少不要虧手續費」 |
| **追蹤止損** | 先漲超過 3%，從最高點回落 1.5% | 「跟著漲，但跌太多就跑」 |
| **最長持有** | 持有超過 15 個交易日 | 「不管賺賠，時間到就出場」 |

> 這些機制的目的是：**賺的時候讓它跑，虧的時候趕快停。**

## 功能二：AI 回測模擬（看過去表現）

### 在哪裡？

模擬交易頁面往下滑，找到「AI 回測模擬」區塊，點擊展開。

### 這是幹嘛的？

你可能會想：「AI 說它很厲害，但我怎麼知道是不是真的？」

回測就是讓 AI **用過去的歷史資料重新跑一遍**，告訴你「如果 30 天前開始跟著 AI 做，你的 100 萬會變多少」。

這就像是考試前先寫模擬考題，看看自己的實力在哪裡。

### 怎麼操作？

1. 展開「AI 回測模擬」區塊
2. 選擇天數（30 天 / 60 天 / 90 天）
3. 選擇模式：
   - **精選標的**：只掃描 50 支大型股，速度快（約 30 秒）
   - **全市場掃描**：掃描 2,000+ 支股票，更真實（約 1~2 分鐘）
4. 點擊「全市場 30 日回測」按鈕
5. 等待結果

### 回測結果怎麼看？

結果出來後會顯示一堆數字，重點看這幾個：

| 看什麼 | 意思 | 怎樣算好 |
|--------|------|---------|
| **總報酬率** | 100 萬變成多少 | 正數就是賺，越高越好 |
| **勝率** | 幾成交易是賺的 | 50% 以上算不錯 |
| **最大回撤** | 過程中最大虧損幅度 | 越小越好，代表風險控制得好 |
| **盈虧比** | 平均賺 vs 平均虧 | 大於 1 代表「賺的比虧的多」 |
| **vs 0050** | 跟直接買 0050 ETF 比 | 勝過 0050 = AI 比被動投資強 |

> **0050 是什麼？** 元大台灣 50 ETF，一次買進台灣前 50 大公司的股票。很多理財書會建議新手直接買 0050，所以拿它當作對照組。

### 歷史回測的實際績效（截至 2026 年 3 月）

| 回測期間 | 報酬率 | 勝率 | 盈虧比 | 同期 0050 |
|---------|--------|------|--------|----------|
| 近 30 天 | **+8.40%** | 55.6% | 2.26 | +8.72% |
| 近 90 天 | -0.72% | 28.2% | 0.96 | -0.42% |

> 近 30 天市場是多頭（漲），AI 賺得跟 0050 差不多。  
> 近 90 天市場偏空（跌），AI 勉強持平、比亂買好。  
> 結論：**AI 在多頭市場表現好，在空頭市場能控制虧損。**

### 重要注意事項

回測跟實際操作可能有差異，因為：
- 回測用的是純數學指標（免費、快速），實際 AI 還會加上新聞分析和 AI 判斷
- 歷史不代表未來，過去賺不代表以後一定賺
- **回測的用途是「參考」，不是「保證」**

## 功能三：智慧選股（分析特定股票）

### 在哪裡？

上方導覽列 → 點擊「智慧選股」

### 這是幹嘛的？

假如你聽朋友說「台積電最近好像不錯」，或是新聞在講「鴻海要漲了」，你想知道：

- 現在到底適不適合買？
- 如果要買，應該怎麼設停損停利？

這時候就輸入股票代號，讓 AI 幫你做一份完整健檢報告。

### 怎麼操作？

1. 在搜尋框輸入股票代號（例如 `2330`）
2. 點擊「開始調研」
3. 等待約 30~60 秒（四位 AI 探員逐一分析）
4. 查看結果

### 結果畫面怎麼看？

**左上大區塊：**
- 股票名稱和代號
- AI 最終判定：**BUY（買）/ HOLD（觀望）/ SELL（賣）**
- AI 信心百分比：越高代表 AI 越確定

**右邊四個小方塊：**

| 方塊 | 意思 | 舉例 |
|------|------|------|
| 進場參考價 | 現在的股價 | $1,890 |
| 止盈目標價 | 漲到這裡可以賣掉獲利 | $1,916 |
| 止損防護價 | 跌到這裡應該認賠賣掉 | $1,789 |
| 建議倉位比例 | 建議你拿幾成資金去買 | 10% |

> **倉位比例怎麼理解？** 如果你有 100 萬可以投資，建議 10% 就是建議你拿 10 萬去買這支。不要把所有錢都壓在同一支股票上。

**下方三份詳細報告（可展開）：**

1. **技術面報告**：從走勢圖的角度分析
2. **情緒面報告**：從新聞和市場氣氛分析
3. **風控面報告**：從資金安全的角度分析

每份報告都是用中文寫的完整分析文章，AI 會解釋「為什麼」做出這個判斷。

### 快捷鍵

搜尋框下方有幾個常用股票按鈕，點一下就能快速分析：

- 2330 台積電
- 2317 鴻海
- 2454 聯發科
- 2412 中華電
- 0050 元大台灣50
- 2603 長榮

### 最重要的提醒

> **AI 給的是「建議」，最終決定權在你手上！**
>
> 就像去看醫生，醫生會給你診斷和建議，但要不要吃藥、開刀，最終是你自己決定。
>
> **股市有風險，沒有任何工具可以保證 100% 賺錢。**

## 如何安裝和啟動

### 需要準備什麼？

| 項目 | 說明 | 哪裡取得 |
|------|------|---------|
| Python 3.11+ | 後端程式語言 | [python.org](https://www.python.org/downloads/) |
| Node.js 18+ | 前端程式語言 | [nodejs.org](https://nodejs.org/) |
| Gemini API Key | Google AI 的鑰匙（免費） | [Google AI Studio](https://aistudio.google.com/) |
| FinMind Token | 台股資料來源的鑰匙（免費） | [FinMind](https://finmindtrade.com/) |

### 第一步：下載程式碼

```bash
git clone https://github.com/RayzChang/APLHATW.git
cd APLHATW
```

### 第二步：安裝套件

```bash
# 安裝 Python 後端套件
pip install -r requirements.txt

# 安裝前端套件
cd frontend
npm install
cd ..
```

### 第三步：設定金鑰

在專案根目錄建立 `.env` 檔案，內容如下：

```
GOOGLE_GEMINI_API_KEY=你的Gemini金鑰貼在這裡
FINMIND_TOKEN=你的FinMind金鑰貼在這裡
```

> **金鑰怎麼申請？**
>
> **Gemini**：到 [Google AI Studio](https://aistudio.google.com/)，用 Google 帳號登入，點「Get API Key」→「Create API key」，複製那串英數字。
>
> **FinMind**：到 [FinMind](https://finmindtrade.com/)，註冊帳號，在「使用者資訊」頁面找到 Token，複製貼上。

### 第四步：啟動

開兩個終端機（命令列視窗）：

**終端機 1 — 啟動後端：**
```bash
uvicorn api.app:app --host 0.0.0.0 --port 8000 --reload
```

看到 `Application startup complete` 就是成功了。

**終端機 2 — 啟動前端：**
```bash
cd frontend
npm run dev
```

看到 `Local: http://localhost:5173/` 就是成功了。

### 第五步：打開瀏覽器

在瀏覽器輸入 `http://localhost:5173`，就能看到 AlphaTW 的畫面了！

## 常見問題

### Q：我的真錢會被動到嗎？

**不會！** 目前系統是純模擬，所有金額都是虛擬的。你的銀行帳戶、證券帳戶完全不受影響。

---

### Q：回測按下去顯示「Network Error」？

1. 確認後端有在執行（終端機 1 有看到 `Application startup complete`）
2. 如果 AI 正在執行全市場掃描，回測可能需要等掃描完成才能使用
3. 重新整理頁面試試

---

### Q：回測跑出來交易次數是 0？

可能原因：
1. **FinMind Token 沒設定**：到 `.env` 檔案確認有填入 Token
2. **Token 額度用完**：FinMind 免費版每小時 600 次，等一小時後再試
3. **市場環境不佳**：如果全市場都在跌，AI 可能判斷「全部都不值得買」，這是正確的行為

---

### Q：AI 分析一次要花多少錢？

| 功能 | 費用 |
|------|------|
| 回測 | 完全免費（純數學計算） |
| AI 全市場掃描（一次） | 約 0.06 美元（台幣約 2 元） |
| 智慧選股分析（一支） | 約 0.008 美元（台幣約 0.25 元） |
| 一個月正常使用 | 約 2~3 美元（台幣約 60~90 元） |

---

### Q：台股什麼時候開盤？

- **週一到週五**，早上 **9:00** 到下午 **1:30**
- 六日休市、國定假日休市
- AI 自動掃描時間：**9:10** 和 **12:30**

> 休市期間，AI 不會執行交易。你可以使用智慧選股和回測功能，但報價會顯示上一個交易日的收盤價。

---

### Q：AI 模擬交易和智慧選股有什麼不同？

| | AI 模擬交易 | 智慧選股 |
|---|---|---|
| **誰挑股票** | AI 自己掃描全市場 | 你指定要分析哪一支 |
| **誰做決定** | AI 完全自主決定買賣 | AI 給建議，你自己決定 |
| **會不會下單** | 會（虛擬模擬單） | 不會，只出分析報告 |
| **適合誰** | 想無腦跟單的人 | 想自己研究但需要 AI 幫忙的人 |

---

### Q：我應該先用哪個功能？

建議的使用順序：

```
1. 先跑「回測」→ 看 AI 的歷史表現，建立信心
        ↓
2. 開啟「AI 模擬交易」→ 觀察幾天，看 AI 怎麼操作
        ↓
3. 用「智慧選股」分析你有興趣的股票 → 練習看懂 AI 的分析報告
        ↓
4. 等你信任 AI 了 → 用自己的證券帳戶，跟著 AI 的決策買賣
```

> **千萬不要一開始就拿真錢跟！** 先觀察至少 1~2 週，確認 AI 的判斷跟你的預期相近，再考慮跟單。
